# SchützenManager (Rework)

Neuentwicklung der Schießsportverwaltung als Cross-Platform-Anwendung (siehe Roadmap in der Projekt-Historie).

Geplanter Stack: Tauri + React/TypeScript (Frontend/Desktop-Hülle), Node.js + Fastify + Prisma (Backend, SQLite lokal / Postgres zentral gehostet).

Der bestehende Java/JavaFX-Code im Hauptverzeichnis bleibt als Referenz für die fachliche Logik erhalten, bis Feature-Parität erreicht ist.
